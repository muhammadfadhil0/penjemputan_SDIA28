# Alarm Sounds

This folder contains alarm sound files for the pickup reminder feature.

## Required Files

Place the following MP3 files in this folder:
- `alarm_classic.mp3` - Classic alarm sound
- `alarm_digital.mp3` - Digital beep sound  
- `alarm_gentle.mp3` - Gentle chime sound

Note: If sound files are not present, the alarm will still work visually but without audio.
